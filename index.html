🦖 RAptor - Créateur de Réalité Augmentée Éducative

Comment utiliser RAptor ?

  1. Sélectionnez un type de média (3D, image, vidéo, son).
  2. Chargez un fichier local ou entrez une URL.
  3. Choisissez un marqueur (ou utilisez le marqueur Hiro par défaut).
  4. Cliquez sur "Lancer la RA" pour prévisualiser.
  5. Exportez votre projet pour obtenir un QR Code à scanner.
