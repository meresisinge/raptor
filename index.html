RAptor AR

Outil de réalité augmentée éducative inspiré de 8th Wall

Mode d'emploi

  1. Sélectionnez un type de média (3D, image, vidéo, son).
  2. Chargez un fichier ou entrez une URL.
  3. Choisissez un marqueur (ou utilisez Hiro par défaut).
  4. Cliquez sur "Lancer la RA" pour prévisualiser.
  5. Exportez votre projet pour obtenir un QR Code.
